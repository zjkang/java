----------------------------------------------------------------------------
导航类快捷键 Navigation shortcut 

Ctrl + F                         : look up in file
Ctrl + Shift + F                 : look up in project (global search)
Shift + Shift                    : pop up search window

Ctrl + N                         : go to Class
Ctrl + Shift + N                 : go to File
Alt + up/down arrow              : method jump in one class
Ctrl + F12                       : display all the methods in the class

Ctrl + G                         : go to line row:column
Ctrl + Home                      : go to first row
Ctrl + End                       : go to bottom row

Ctrl + Alt + left/right arrow.   : jump back/forth previous cursor position
Ctrl + E                         : Looks up historical open class

Ctrl + B                         : jump to definition for class or jump to where interface method are called
Ctrl + Alt + B                   : what implementation class use the interface method


----------------------------------------------------------------------------
文件操作类 shortcut

From interface: quick generate impl class and unit test class
*Step: Click on interface name -> Alt + Enter -> select implment the interface
*Step: Click on interface name -> Ctrl + Shift + T -> select unit test version and create unit test


----------------------------------------------------------------------------
辅助编码 shortcut

Ctrl + /                               : single line comment
Ctrl + Shift + /                       : block comment
/** <Enter>                            : doc comment on top of the method

Ctrl + Alt + L                         : format the code
Alt + Insert                           : add getter/setter, constructor, hash, equals, toString
Ctrl + Shift + Enter                   : autocomplete
Ctrl + Shift + Space                   : show tips

Ctrl + Alt + T                         : create surround for code black (try catch, if, while) 
Ctrl + P                               : method parameter tip

Ctrl + I                               : class from interface/abstract class implement method override abstract
Ctrl + O                               : class from abstract class implement other methods

Ctrl + D                               : copy one row
Ctrl + X                               : cut one row
Ctrl + Y                               : delete one row
Ctrl + shift + (up/down arrow)         : move code block

Setting -> search auto -> Auto Import -> Optimize import on the fly

Ctrl + W                               : expand selection
Ctrl + Shift + W                       : reduce selection

Alt + Enter                            : import package
Alt + Enter                            : introduce new local variable


--------------------------------------------------------------------------
重构快捷键

Shift + F6                             : 变量重命名
Ctrl + R                               : current file replace
Ctrl + Shift + R                       : global replacement
Ctrl + Alt + V                         : expression replacement to variable
Ctrl + Alt + C                         : expression replacement to constant
Ctrl + Alt + M                         : 一段代码提取成一个方法
